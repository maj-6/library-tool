# Bundled fonts

- `roboto-slab-var.woff2` — Roboto Slab (variable weight, latin subset, v36)
- `roboto-slab-var-latin-ext.woff2` — the latin-ext subset of the same face

Both from Google Fonts (fonts.gstatic.com), declared in `../site.css` with the
matching unicode-ranges so a plain-ASCII page downloads one file. Licensed
under the SIL Open Font License 1.1
(https://github.com/googlefonts/robotoslab). Bundled locally so the site keeps
its no-CDN promise.
