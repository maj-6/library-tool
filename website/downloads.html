Building from source

The desktop app is a Flask backend in an Electron shell; the capture app is an Android Studio project. Both build from the repository — desktop/README.md and android/BookCapture/README.md have the steps.

What the desktop app needs

Tesseract, if you want to OCR image-only scans. API keys, if you want cloud OCR or the phone-capture pipeline. Neither is required to catalogue, check copyright, or read a scan that already carries a text layer. Everything is optional and everything is entered in Settings.
